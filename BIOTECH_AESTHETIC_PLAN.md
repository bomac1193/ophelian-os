## Neural State (Replaces Psychological State)

### **Frontend Terminology:**

**CHARGE:** -10 (tranquil) ↔ +10 (intense)
- Replaces: Hot/Cool axis (-1 to +1)
- Maps directly: Charge / 10 = Hot/Cool value

**STABILITY:** 0% (chaotic) ↔ 100% (integrated)
- Replaces: Shadow Integration + Individuation
- Average of both backend values

**PHASE:**
- Emergence → Ascent → Crisis → Descent → Integration → Transcendence
- Direct 1:1 mapping to backend Trajectory

## Chromox Integration for Visual Generation

**Auto-generate character visuals** based on genome:

```typescript

const primitive = imprint.primitive; // Cube ⬡
const charge = character.genome.charge; // +3
const phase = character.genome.phase; // Integration

const chromoxPrompt = `
Biotech character portrait: ${character.name}
Classification: ${imprint.symbol}-${imprint.name}
Geometric signature: ${primitive}
Aesthetic: Industrial precision, brushed metal, geometric grids
Color palette: Metallic silver, electric blue, sharp edges
Charge state: Moderate intensity (+3), structured energy
Phase: Integration - balanced, focused, purposeful
Style: Futuristic bioengineering lab, Matrix-inspired, clean sci-fi
Include: ${primitive} symbol subtly integrated into design
`;

// Call Chromox API → Get image URL
```
